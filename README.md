## web3.js - Ethereum JavaScript API

web3.js是一个库集合，允许您使用HTTP或IPC连接与本地或远程以太它节点进行交互。

以下文档将指导您[安装和运行web3.js](https://web3js.readthedocs.io/en/1.0/getting-started.html#adding-web3)，以及提供带有示例的API参考文档。
